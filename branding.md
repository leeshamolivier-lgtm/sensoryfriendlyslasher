# Podcast Branding

## Color Palette

This podcast uses a dark mode layout built around the following core color palette:

* **Deep Charcoal**: `#1A1A1D`
  * Used for the main background to create a sleek, immersive dark mode experience.
* **Soft Cream**: `#F7F4EA`
  * Used for primary text and major headings, providing excellent contrast against the dark background while being softer on the eyes than pure white.
* **Muted Red**: `#A94047`
  * Used for accents, buttons, active states, and highlights to add a pop of color without being overwhelmingly bright.
